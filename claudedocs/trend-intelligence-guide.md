## 1. 개요

### 트렌드 인텔리전스란?

트렌드 인텔리전스는 소셜 미디어(주로 TikTok)에서 **떠오르는 트렌드를 자동으로 발견하고 분석**하여, 콘텐츠 크리에이터와 브랜드가 **데이터 기반의 콘텐츠 전략**을 수립할 수 있도록 돕는 AI 시스템입니다.

### 비유로 이해하기

| 비유 | 일반 검색 | 트렌드 인텔리전스 |
|------|----------|-----------------|
| 금 채굴 | 이미 발견된 금광에서 채굴 | 아직 아무도 모르는 금맥 탐사 |
| 쇼핑 | 인기 상품 목록 보기 | 다음 주에 품절될 상품 미리 알기 |
| 주식 | 현재 주가 확인 | 오를 종목 예측 |

## 2. 핵심 개념

### 2.1 트렌드 탐색 (Trend Exploration)

하나의 키워드에서 시작해 **연관된 새로운 키워드를 자동으로 발견**하는 과정입니다.

```
[탐색 과정 예시]

입력: "K-pop"
    │
    ▼
1단계: TikTok에서 K-pop 관련 영상 수집
    │
    ▼
2단계: 해시태그 추출 (#bts, #newjeans, #kpopidol 등)
    │
    ▼
3단계: 각 해시태그에서 또 다른 해시태그 발견
    │
    ▼
4단계: 3단계까지 확장 탐색 반복
    │
    ▼
출력: 아직 대중화되지 않은 "떠오르는" 키워드 목록
```

### 2.2 트렌드 분석 (Trend Analysis)

발견된 트렌드가 **왜 뜨는지, 어떻게 활용해야 하는지** AI가 분석합니다.

| 분석 유형 | 분석 내용 |
|----------|----------|
| 텍스트 분석 | 효과적인 해시태그 조합, 인기 캡션 패턴 |
| 영상 분석 | 잘 먹히는 영상 스타일, 효과, 구도 |
| 종합 리포트 | 최적 업로드 시간, 타겟 오디언스, 경쟁 현황 |

### 2.3 트렌드 확장 (Trend Expansion)

하나의 트렌드에서 **관련된 크리에이터, 해시태그 네트워크**를 발견합니다.

- 함께 자주 쓰이는 해시태그 조합 발견
- 해당 트렌드를 잘 활용하는 크리에이터 추천
- 추가 탐색을 위한 키워드 제안

## 3. 주요 기능

### 3.1 키워드 탐색

```
┌─────────────────────────────────────────────────┐
│  입력: 시드 키워드 (예: "dance challenge")        │
│                     │                           │
│                     ▼                           │
│  ┌─────────────────────────────────────────┐   │
│  │         TikTok 영상 검색 및 수집          │   │
│  └─────────────────────────────────────────┘   │
│                     │                           │
│                     ▼                           │
│  ┌─────────────────────────────────────────┐   │
│  │         해시태그 추출 및 점수 계산         │   │
│  └─────────────────────────────────────────┘   │
│                     │                           │
│                     ▼                           │
│  ┌─────────────────────────────────────────┐   │
│  │         3단계 깊이까지 확장 탐색          │   │
│  └─────────────────────────────────────────┘   │
│                     │                           │
│                     ▼                           │
│  출력: 신선도/인기도 점수가 매겨진 키워드 목록    │
└─────────────────────────────────────────────────┘
```

### 3.2 텍스트 트렌드 분석

- **해시태그 클러스터링**: 의미별로 해시태그 그룹화
- **캡션 템플릿**: 인기 있는 캡션 패턴 분석
- **해시태그 전략**: 메인/서브/니치 해시태그 추천
- **감성 분석**: 트렌드의 전반적인 분위기 파악

### 3.3 영상 트렌드 분석

- **비주얼 스타일**: 인기 있는 영상 스타일 분석
- **효과 추천**: 자주 사용되는 효과, 필터
- **컬러 팔레트**: 트렌드에 맞는 색상 조합
- **기술 스펙**: 권장 화면비, 영상 길이, 카메라 앵글

### 3.4 종합 리포트

- **트렌드 점수**: 전체적인 트렌드 건강도 (0~100)
- **트렌드 방향**: 상승/유지/하락 예측
- **최적 업로드 시간**: 데이터 기반 추천
- **타겟 오디언스**: 주요 시청자층 분석
- **경쟁 분석**: 콘텐츠 갭과 기회 발견

## 4. 점수 시스템

### 4.1 신선도 점수 (Novelty Score)

**"얼마나 새로운 트렌드인가?"** (0~100점)

| 점수 범위 | 의미 |
|----------|------|
| 80~100 | 매우 새로움 - 아직 대부분이 모르는 트렌드 |
| 60~79 | 새로움 - 일부만 알고 있는 트렌드 |
| 40~59 | 보통 - 어느 정도 알려진 트렌드 |
| 0~39 | 익숙함 - 이미 널리 알려진 트렌드 |

**신선도 점수 계산 요소:**
- 사용자의 이전 검색/탐색 이력
- 시드 키워드로부터의 거리 (멀수록 높음)
- 최초 발견 시점 (최근일수록 높음)
- 전체 사용자 중 희귀도

### 4.2 인기도 점수 (Popularity Score)

**"얼마나 반응이 좋은 트렌드인가?"** (0~100점)

| 점수 범위 | 의미 |
|----------|------|
| 80~100 | 폭발적 - 조회수, 좋아요 매우 높음 |
| 60~79 | 인기 - 안정적인 참여율 |
| 40~59 | 보통 - 평균적인 성과 |
| 0~39 | 낮음 - 아직 검증되지 않음 |

**인기도 점수 계산 요소:**
- 평균 참여율 (좋아요 + 댓글 + 공유 / 조회수)
- 총 조회수
- 출현 빈도
- 최근 활동량

### 4.3 점수 활용 전략

| 전략 | 신선도 | 인기도 | 적합한 상황 |
|------|-------|-------|-----------|
| 선점 전략 | 높음 | 낮음~중간 | 얼리어답터, 트렌드 리더 |
| 안정 전략 | 낮음~중간 | 높음 | 확실한 성과가 필요할 때 |
| 균형 전략 | 중간 | 중간 | 리스크와 기회의 균형 |

### 예시 1: 댄스 콘텐츠 크리에이터

│ 검증된 키워드 (인기도 높음)                      │
│    • #tiktokdance (신선도 25, 인기도 92)        │

├────────────────────────────────────────────────┤
│ 영상 스타일 추천                                │

│    • 평일: 저녁 8~10시                         │
│    • 주말: 오후 2~4시                          │
├────────────────────────────────────────────────┤
│ 참고 크리에이터                                 │

## 7. 일반 검색과의 차이점

| 구분 | 일반 검색 | 트렌드 인텔리전스 |
|------|----------|-----------------|
| **결과** | 지금 인기 있는 것 | 앞으로 인기 있을 것 |
| **탐색 깊이** | 1단계 (검색어만) | 3단계 (연관 키워드까지) |
| **분석** | 없음 | AI가 왜 뜨는지 분석 |
| **추천** | 없음 | 콘텐츠 제작 가이드 제공 |
| **개인화** | 없음 | 사용자 이력 기반 맞춤 추천 |
| **예측** | 불가능 | 트렌드 방향 예측 |
