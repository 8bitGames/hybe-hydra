# HYDRA UX Task-Based Usability Test Plan

**Version**: Post-Redesign (4-nav structure)
**Test Duration**: 30-45 minutes per participant

---

## Test Setup

### Participants

| Type | Count | Criteria |
|------|-------|----------|
| New Users | 4 | Never used HYDRA before |
| Existing Users | 4 | Used previous 6-nav version |

### Environment

- Browser: Chrome (latest)
- Screen recording: Enabled
- Think-aloud protocol: Encouraged
- Facilitator: Non-leading prompts only

### Pre-Test Checklist

- [ ] Test account created with sample data
- [ ] Screen recording software running
- [ ] Participant consent form signed
- [ ] Browser cleared of cache/cookies
- [ ] Sample campaigns pre-created for existing user tasks

## Task Scenarios

### Scenario 1: First Video Creation (New Users)

**Goal**: Create a video using AI generation from scratch

**Briefing Script**:
> "You want to create a short video for social media. The video should show a country music artist performing on stage. Please find where to start and create this video."

**Success Criteria**:
- [ ] Reaches generation page within 3 clicks
- [ ] Completes generation request
- [ ] Time: Under 3 minutes

## Post-Test Interview Questions

### For New Users

1. What was your first impression of the Home page?
2. Did you feel you knew where to start?
3. What would you change about the navigation?
4. Was anything confusing about the terminology?
5. Would you prefer more or fewer menu options?

### For Existing Users

1. What do you miss from the old navigation?
2. What improvements do you notice?
3. How long do you think it would take to adjust?
4. Would you recommend keeping or reverting changes?
5. What specific features became harder to find?
